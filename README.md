# Real-Time Crypto Trading Bot (Node + React)

- Backend: `server` (Express, Node 18+ fetch)
- Frontend: `app` (Vite + React)

Run on Windows: double-click `Start Bot.bat`.

Manual run:
- Terminal 1:
  - `cd server`
  - `npm install`
  - `npm run dev`  (API at http://localhost:5174)
- Terminal 2:
  - `cd app`
  - `npm install`
  - `npm run dev`  (UI at http://localhost:5173)

Build:
- `cd server && npm run build`
- `cd app && npm run build`
